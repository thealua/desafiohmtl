Tudo que Presica Saber Sobre HTML

Você sabia que o HTML não é uma linguagem de programção, afinal o que é o HTML?

Primeiro devemos entedender que uma liguagem de programação é como o programador se comunica com a máquina. Então o que é o HTML?

O HiperText Markup Language - linguagem de marcação de hipertexto. Um hipertexto nada mais é que um texto que é usado para fazer referência a outros textos, com ele é possível criar a estrutura, definir o posicionamento dos elementos, estruturar a divisão de uma página em blocos visuais e combinar texto, imagens e vídeos. Resumindo a linguagem de marcação é o que diz para o servidor web qual o estilo e estrutura do documento

O HTML pode ser usado para desenvolvimento web, navegação na internet, documentação e outras coisas. Além de que agora o HTML é considerado padrão oficial da internet, o que isso significa? Que a World Wide Consortium (W3C) mantém e desenvolve especificações e atualizações regulares. Então continuaremos usando o HTML sem medo de ser feliz, pois não ficará obsoleto tão cedo.

O básico que funciona

 Voltar

O HTML tem uma estrura básica para funcionar. Nesse tópico iremos falar mais sobre isso!

Pra começar existe algumas denominações para identificar os diferentes itens da estrutura do HTML. E esses comandos são tags, elementos e o atributos. As tags representa denominação de um elemento, é o que diz para o navegador onde começa e termina, é um rótulo para indicar qual a função daquele conteúdo. O atributo representa as caracteristicas e as funcionalidades que cada elemento pode ter; a atributos globais que funcionam igual para todos os elementos e atributos específicos. O elemento representa um item presente na página, como um parágrafo, uma imagem, um link, um formulário e outros recursos da página.

    As três principais partes de um elemento:
  • Tag de Abertura
  • Conteudo
  • Tag de Fechamento

Funciona basicamente assim, a tag de abertura diz onde o elemento começa a ter efeito, ela tem colchetes angulares para abertura e fechamento assim: < >, dentro dela é atribuido '*'. A tag de fechamento é igual com a diferença de que entre os colchetes vem uma barra, assim: < / '*' >. Entre elas vem o CONTEUDO, a parte que os usuários verão.

O atributo é o que fica dentro da tag, é essa informação que controla o comportamento do elemento da maneira que esperamos. Os atributos mais básicos são:

  • A tag <html> é o elemento raiz que define todo o documento HTML.
  • A tag <head> contém metadados sobre o título e a codificação de caracteres da página.
  • A tag <body> inclui todo o conteúdo que aparece na página.
  • A tag <h1> cria título principal. Títulos tem hierarquia que vai do h1 até o h6.
  • A tag <p> é para adicionar um parágrafo no HTML
  • A tag <ol>, e a tag <ul> são listas que podem ser ordenadas ou desordenada. A tag <li> é usada para representar um item dentro dessas listas.
  • A tag <u> representa sublinhar enquanto a tag <i> é o itálico, e o <strong> o negrito.

Umas paradas legal pra saber

 Voltar

Uma coisa interessante sobre o HTML é que é uma linguagem muito básica, e mesmo utilizando outras linguagens de programação resultará em códigos HTML.

  • O HTML não precisa de ferramentas avançadas para codar, um simples bloco de notas ja é o suficiente, é importante também para fixar os códigos de forma mais rápida. Com tempo Você poderá aderir a outros editores, se assim desejar!
  • Quando estamos iniciando e você olha para um código-fonte é de assustar qualquer um, porém não devemos olhar para ele como se fosse impossivel. O código-fonte deve ser olhado por partes, linha por linha, trecho por trecho; e não como um todo, assim nao será mais tão assustador!!
  • O DOCTYPE é o que renderiza o site, você não pode deixar passar. O DOCTYPE é uma declaração que informa para o navegador o tipo de documento e qual é a versão do HTML que ele está usando. A versão 5 o HTML deixou de ser baseado em SGML (Standard Generalized Markup Language), e com isso a forma de declarar o DOCTYPE foi simplificada.
  • Nunca esqueça de informar a codificação de caracteres no HTML, sem isso vai ficar uma bagunça na hora de reproduzir o código. Isso acontece porque o HTML não consegue, sozinho, interpretar todos os simbolos que existem no mundão a fora! Então através do meta tag CHARGET informamos o tipo de codificação. No caso do Brasil usamos o <meta charset=”UTF-8″> para usar os 128 caracteres básicos
